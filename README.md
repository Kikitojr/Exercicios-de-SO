Este repositório reúne exercícios desenvolvidos durante a disciplina de Sistemas Operacionais, no curso de Análise e Desenvolvimento de Sistemas pela FATEC Zona Leste, ministrada pelo professor Leandro Colevati.

As atividades têm como base exercícios propostos originalmente na disciplina de Lógica de Programação, pelo professor Ricardo Satoshi, inicialmente estruturados em pseudocódigo e posteriormente implementados em Python, com o objetivo de reforçar a lógica de programação e sua aplicação em um ambiente real de execução.

O desenvolvimento dos exercícios é realizado em ambiente Windows, utilizando Docker para execução dos códigos em containers, com uso de pastas compartilhadas (volumes) para integração entre o sistema host e o ambiente isolado. Essa abordagem permite praticar conceitos importantes como virtualização, gerenciamento de containers, uso de terminal e organização de projetos em ambientes controlados.

O repositório registra a evolução ao longo das atividades propostas na disciplina, incluindo a conversão de algoritmos para código executável, execução em containers, manipulação de arquivos via linha de comando e utilização de ferramentas comuns no fluxo de desenvolvimento.

Este material faz parte do meu processo de formação acadêmica e prática em desenvolvimento de software, com foco no fortalecimento dos fundamentos de programação, no entendimento de sistemas operacionais e no uso de tecnologias amplamente utilizadas no ambiente profissional.
